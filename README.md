DeepShield – Gemini Integration Prototype
📌 Overview
DeepShield is a government-grade detection platform designed to identify deepfakes and AI-generated media.
This version replaces the Sightengine API with Google Gemini for multimodal detection.

🚀 Features
Upload Media: Drag & drop images/videos.

Detection Engine (Gemini): Analyzes files for authenticity, deepfake manipulation, and AI generation.

Case Management: Logs detections with case ID, severity, confidence, and status.

Reporting Module: Secure, anonymized reports to cyber crime authorities.

Evidence Chain: SHA-256 tamper-proof verification for legal admissibility.
